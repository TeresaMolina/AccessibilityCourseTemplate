How to Use in Brightspace

Use an HTML Template (Content Area)

  1. Go to Content in your course.
  2. Select Create New → HTML Document.
  3. Open Source (the </> icon).
  4. Copy the template HTML from this site and paste it into the editor.
  5. Replace placeholders (titles, dates, links) with your course content.
  6. Save and preview on desktop + mobile.

Tip: Keep headings in order (H2 → H3) and avoid relying on color alone to convey meaning.
